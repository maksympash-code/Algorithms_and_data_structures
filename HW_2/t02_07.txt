f(n) = 3n^2 - n + 4
g(n) = n log(n) + 5

f(n) + g(n) = 3n^2 - n + n log(n) + 9
n^2 росте швидше, ніж n log(n), тому складність ми можемо оцінити, як O(n^2)

також працює властивість із лекції
f1 = O(g1)
f2 = O(g2) тоді
f1 + f2 = O(max[g1,g2])

в нашому випадку
f1 = O(n^2)
f2 = O(n log(n))
f1 + f2 = O(max[n^2, n log(n)]) = O(n^2)
