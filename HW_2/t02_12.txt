def h(n):
    return f(n) + g(n)

f(n) - Складність O(n)
g(n) - Складність O(n^2)

За властивістю
f1 = O(g1)
f2 = O(g2) тоді
f1 + f2 = O(max[g1,g2])

h(n) = O(max[n, n^2]) = O(n^2) - Складність h(n)

f(n) = (n * (n + 1)) // 2
g(n) = sum{n, i = 1}: i + (i * (i + 1)) // 2

Результатом виконання функції h(n) є сума f(n) та g(n)
Його можна оптимізувати, але лише окремі частини при цьому складність не зміниться
