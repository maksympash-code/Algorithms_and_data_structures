def f(n):
    sum = 0                         | O(1)
    for i in range(1, n + 1):       | O(n)
        sum = sum + i               | O(n)
    return sum                      | O(1)

Результатом виконання є обчислення суми перших n натуральних чисел
У найгіршому випадку складність буде O(n), але його можна оптимізувати:

def f(n):
    return (n * (n + 1)) // 2

Оптимізований алгоритм має складність O(1)
